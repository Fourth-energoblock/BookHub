## Основные характеристики проекта

- Название: BookHub - Платформа для социального чтения
- Тип: MVP веб-приложения и Android-приложения
- Целевая аудитория: Студенты, начинающие специалисты
- Технологический стек: React, Node.js (Express), MongoDB, Android (Kotlin)
- Срок разработки: 01.12.2025 - 19.02.2026
- Команда исполнителя: 5 разработчиков ("4 энергоблок")
- Команда заказчика: 4 участника ("Квартет П")

## Функциональность MVP

- Регистрация и авторизация пользователей
- Каталог книг с интеграцией Project Gutenberg, Google Books API, LitRes
- Модуль чтения с настройками (размер шрифта, тема)
- Система аннотаций с тремя уровнями доступа (приватный, публичный, по ссылке)
- Личные коллекции книг (до 10 в бесплатной версии)
- Офлайн-доступ для Android

## KPI успешности

- 1000 регистраций за первые 3 месяца
- Retention Rate более 40%
- Среднее время сессии около 15 минут
- Не менее 500 созданных аннотаций
- Доступность системы не менее 99,9%

## Работа с документом

### Просмотр
Откройте файл `BookHub_TZ.pdf` для просмотра готового документа.

### Редактирование и компиляция

1. Установите Typst:
   ```bash
   brew install typst
   ```

2. Отредактируйте нужные разделы в папке `sections/`

3. Скомпилируйте документ:
   ```bash
   cd BookHub/TZ
   typst compile main.typ BookHub_TZ.pdf
   ```

### Изменение данных проекта
Для изменения данных о проекте и участниках отредактируйте файл `data.typ`.
